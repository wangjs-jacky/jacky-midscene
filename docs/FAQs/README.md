# 常见问题（FAQs）目录

本目录用于记录学习过程中遇到的所有问题和解答。

## 目录结构

建议按以下方式组织文件：

```
docs/FAQs/
├── 环境配置问题.md
├── API使用问题.md
├── 桥接模式问题.md
├── YAML脚本问题.md
├── 测试集成问题.md
└── ...
```

或者按时间顺序：

```
docs/FAQs/
├── FAQ-2024-01.md
├── FAQ-2024-02.md
└── ...
```

## 文件命名规范

- 按主题分类：`环境配置问题.md`、`API使用问题.md`
- 按时间顺序：`FAQ-2024-01.md`
- 使用中文或英文都可以，保持一致性即可

## 文件模板

每个 FAQ 文件应包含以下内容：

```markdown
# 常见问题 - [主题]

## Q1: [问题标题]
**问题描述**：[详细描述问题]

**答案**：[问题的解答]

**相关资源**：
- [相关文档链接]
- [相关代码位置]

**标签**：#标签1 #标签2

---

## Q2: [问题标题]
...
```

## 使用说明

1. 遇到问题时，先记录问题描述
2. 解决问题后，及时补充答案和解决方案
3. 添加相关资源链接和代码位置，方便后续查阅
4. 使用标签分类，方便检索
5. 定期整理和更新 FAQ 文档

## 标签系统

建议使用以下标签分类：

- `#环境配置` - 环境搭建和配置相关问题
- `#API` - API 使用相关问题
- `#桥接模式` - Bridge Mode 相关问题
- `#YAML` - YAML 脚本相关问题
- `#测试` - 测试集成相关问题
- `#错误排查` - 错误排查和调试
- `#最佳实践` - 最佳实践和经验分享

## 搜索技巧

使用标签可以快速找到相关问题：
- 在编辑器中搜索标签，例如：`#环境配置`
- 使用文件搜索功能查找关键词
